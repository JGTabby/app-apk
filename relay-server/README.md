# 喵喵連接網路中繼（測試用）

這個中繼讓兩支手機透過 Wi-Fi 或行動數據互傳資料。兩台手機都必須能連到一台公開可存取的伺服器。

## 啟動

在有 Node.js 的伺服器執行：

```powershell
node server.js
```

預設 TCP 連接埠是 `7000`。雲端防火牆、伺服器防火牆與路由器都必須允許這個 TCP 連接埠。不要把家用電腦直接暴露到公開網路；測試時建議改用受信任 VPN 或具 TLS 保護的中繼。

## App 設定

1. 在兩支手機開啟「啟用網路中繼測試模式」。
2. 填入相同的主機、連接埠與至少 6 字元的配對代碼。
3. 兩邊都按「連接網路中繼」。

這是實驗功能：中繼只負責轉傳，尚未提供端對端加密、帳號驗證或背景持續連線。請勿用於敏感剪貼簿內容或精確位置資料。
